# 🌲 EcoType: Forest Cover Type Classification

This project predicts forest cover types using machine learning based on environmental features.

---

## 🚀 Objectives

- Perform data cleaning, preprocessing, and EDA  
- Build multiple ML models and evaluate them  
- Tune the best model  
- Deploy the prediction app using **Streamlit**
